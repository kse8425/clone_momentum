# Momentum Clone Coding

## 개요

NomadCorders JavaScript 강의를 수강해 구글 확장 프로그램 Momentum을 클론 코딩했습니다.

DB대신 브라우저의 LocalStorage를 사용했고, 배경화면을 unsplash에서 받아와 랜덤으로 변경되게 수정했습니다.

## 기능

- 현재 시간 출력
- 사용자 이름 저장
- ToDo 리스트 추가, 삭제
- 현재 지역 날씨 조회
- 랜덤 배경화면
